# MemoryNode Next.js Agent (Phase-1)

Minimal Next.js app that ingests a note and searches it via the MemoryNode JS SDK.

## Env vars
- `NEXT_PUBLIC_MEMORYNODE_API_KEY`
- `NEXT_PUBLIC_MEMORYNODE_OWNER_ID`

## Run
```bash
npm install
npm run dev
```

Base URL is fixed to `https://worker.memorynode.ai`.
